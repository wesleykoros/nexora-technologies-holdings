# Nexora Technologies Holdings LLC Website

Premium corporate website built with Next.js 16, React 19, Tailwind CSS 4, Motion, Lucide and Resend.

## Local setup

```powershell
Copy-Item .env.example .env.local
npm install
npm run dev
```

Open http://localhost:3000.

## Production checks

```powershell
npm run lint
npm run build
```

## Deployment

Import the repository into Vercel, add the variables from `.env.example`, and connect `nexoratechholdings.com`.

The privacy policy and terms are working drafts and should be reviewed by qualified U.S. counsel before publication.
